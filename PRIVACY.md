## 1. 本拡張機能について

DriveWebPreviewer（以下「本拡張機能」）は、ユーザーの Google Drive 上にある HTML/CSS/JavaScript/画像等で構成された Web ページを、ブラウザ内でそのままプレビュー表示する Chrome 拡張機能です。

## 2. 取得・利用する情報

本拡張機能は、プレビュー表示の目的に限り、以下を**ユーザーのブラウザ内でのみ**扱います。

- **Google Drive のファイル内容・メタデータ（ファイル名・親フォルダ・更新日時等）**: ユーザーがプレビューを指示したファイル、およびその表示に必要な同一サイト内の関連ファイルを Google Drive API から取得します。スコープは閲覧用（`drive.readonly`）です。
- **OAuth アクセストークン**: Google へのアクセスに用います。`chrome.identity` が管理し、ブラウザ内に保持されます。

## 3. 情報の送信・第三者提供

- 本拡張機能は、取得したファイル内容・メタデータ・アクセストークンを、**開発者や第三者のサーバーへ送信しません**。通信先は Google の API（`googleapis.com` 等）に限られます。
- 取得データを販売・共有・広告目的で利用することはありません。

## 4. 保存・キャッシュ

- 表示を高速化するため、取得したファイルをブラウザの Cache Storage に**一時的にキャッシュ**します。これはユーザーのデバイス内に留まり、設定画面からいつでも削除できます。
- アクセストークンを永続ストレージに平文保存することはありません。

## 5. 権限の利用目的

- `identity`: Google へのサインインとアクセストークン取得
- `storage`: プレビューセッション状態・設定の保存（デバイス内）
- ホスト権限 `https://www.googleapis.com/*`: Google Drive API への通信
- `drive.readonly` スコープ: プレビュー対象ファイルの閲覧

## 6. データの削除

- キャッシュ: 設定画面の「キャッシュを削除」で消去できます。
- 連携解除: [Google アカウントのアプリ連携](https://myaccount.google.com/connections) から本拡張機能のアクセスを取り消せます。
