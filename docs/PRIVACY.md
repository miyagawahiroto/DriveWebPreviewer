# プライバシーポリシー（DriveWebPreviewer）

> **草案**。公開前に「連絡先メール」「公開 URL」「事業者名（必要な場合）」を確定し、Web 上に公開して URL を取得すること（OAuth 検証・CASA の必須要件）。Google の審査では英語版も求められる場合があるため、確定後に英訳版の併設も検討する。
>
> 最終更新日: 2026-06-25（公開時に確定）

## 1. はじめに

DriveWebPreviewer（以下「本拡張」）は、利用者の Google Drive 上に置かれた HTML / CSS / JavaScript / 画像で構成された Web ページを、外部サービスにアクセス権限を渡すことなく、ブラウザの拡張機能内で直接レンダリングしてプレビューする Chrome 拡張機能です。

本ポリシーは、本拡張が取り扱う情報と、その取り扱い方針を説明します。

## 2. 取得する情報

本拡張は、利用者が明示的にプレビュー操作を行ったときにのみ、以下を取り扱います。

| 情報 | 取得元 | 用途 |
|------|--------|------|
| Google Drive 上のファイル内容（HTML / CSS / JS / 画像等のバイトデータ） | Google Drive API（利用者の認可に基づく） | プレビュー対象ページのレンダリング表示 |
| ファイルのメタデータ（ファイル名・ID・親フォルダ ID・MIME タイプ・更新日時） | Google Drive API | 相対パス解決・同名ファイルの最新判定 |
| OAuth アクセストークン | Google 認証（`chrome.identity`） | Google Drive API への読み取りリクエスト認可 |

本拡張は、上記以外の個人情報（氏名・メールアドレス・閲覧履歴等）を独自に収集しません。

## 3. 情報の利用目的

取得した情報は、**利用者の Drive 上の Web ページを利用者自身のブラウザ内でプレビュー表示する**目的のみに使用します。

## 4. 情報の保存と保持期間

- **OAuth アクセストークン**: ブラウザ内の `chrome.storage.session`（セッション記憶域）にのみ一時保存します。永続ストレージへ平文保存しません。ブラウザのセッション終了・サインアウトで破棄されます。
- **ファイル内容**: プレビュー高速化のため、ブラウザ内の Cache API にプレビューセッション単位で一時キャッシュします。永続的な外部保存は行いません。
- いずれもブラウザ内に閉じており、利用者の端末外へ送信・保存しません。

## 5. 第三者への提供・外部送信

- 本拡張は、ファイル内容・メタデータ・OAuth トークン・ファイル ID を、**開発者を含むいかなる外部サーバー・第三者にも送信しません**。
- 通信先は **Google の API エンドポイント（`*.googleapis.com`・`accounts.google.com` 等）に限定**されます。
- 解析・広告・テレメトリ目的のデータ収集や外部送信は一切行いません。

## 6. Google ユーザーデータの取り扱い（Limited Use）

本拡張による Google ユーザーデータの取得・利用・移転は、[Google API サービスユーザーデータポリシー](https://developers.google.com/terms/api-services-user-data-policy)（その Limited Use 要件を含む）に準拠します。具体的には：

- 取得した Google ユーザーデータは、本ポリシー 3 に記載した、利用者に明示した機能の提供のみに使用します。
- 当該データを広告目的で利用・移転しません。
- 当該データを人間が読み取ることはありません（利用者本人がプレビュー画面で閲覧する場合、またはセキュリティ・法令遵守のために必要かつ利用者が同意した場合を除く）。
- 当該データを第三者に販売しません。

## 7. 使用する権限とスコープ

- OAuth スコープ: `https://www.googleapis.com/auth/drive.readonly`（**読み取り専用**）。本拡張は利用者の Drive ファイルを変更・削除・作成しません。
- 拡張機能の権限: `identity`（認証）・`storage`（状態保存）、および `https://www.googleapis.com/*` への通信。閲覧画面への注入は `https://drive.google.com/*` に限定されます。

## 8. 利用者の管理（コントロール）

- 本拡張の設定ページからサインアウト（保存済みトークンの破棄）ができます。
- Google アカウント側の連携解除は [Google アカウント > データとプライバシー > サードパーティの接続](https://myaccount.google.com/connections) から行えます。
- 本拡張のアンインストールにより、ブラウザ内に保持された情報は削除されます。

## 9. セキュリティ

- OAuth クライアントシークレットを保持しません（implicit flow）。
- トークンをログ出力・永続平文保存しません。
- 通信は TLS（HTTPS）で行います。
- 詳細な設計方針は本拡張のセキュリティ文書（`SECURITY.md`）に基づきます。

## 10. 児童のプライバシー

本拡張は一般向けの開発・閲覧支援ツールであり、児童を対象としていません。

## 11. ポリシーの変更

本ポリシーは必要に応じて改定されます。重要な変更がある場合は、本ページおよびストア掲載情報で告知します。
