# STORE_LISTING - Chrome ウェブストア掲載文（下書き）

公開時にダッシュボードへ貼る掲載情報の下書き。現在の動作（フォルダ/ファイルを開いて「Web プレビュー」、スコープ `drive.readonly`）に対応。`drive.file` ＋ Picker へ移行した際は「使い方」を更新する。

## 名前
DriveWebPreviewer

## 概要（短い説明・132 文字以内）

```
Google ドライブ上の HTML/CSS/JS で作った Web ページを、外部にアクセス権限を渡さずブラウザ内でそのままプレビュー表示します。
```

## 説明（アイテムの紹介＋インストールのメリット）

```
Google ドライブにアップした HTML / CSS / JavaScript・画像で構成された Web ページを、
ソースコードではなく「本物の Web ページ」としてその場でプレビューできる拡張機能です。

■ こんな方におすすめ
・制作中の LP・モックアップ・静的サイトを、Drive 上で手早く確認したい
・社内資料やデザイン確認を、外部サービスに渡さず安全に共有したい
・ビルド済みの成果物を、サーバーを立てずにサッと表示確認したい

■ インストールするメリット
・そのまま表示：Drive 標準では HTML がソースのまま表示されますが、本拡張は正しく描画します
・リンク切れしない：CSS・JavaScript・画像・サブフォルダの相対パスも自動で解決します
・安全・安心：取得処理はすべてあなたのブラウザ内で完結。ファイルもログイン情報も外部サーバーへ送りません（通信先は Google だけ）
・かんたん：フォルダやファイルを選んでボタンを押すだけ。Google ログインのみで使えます
・Markdown 対応：.md ファイルは読みやすく整形して表示します
・最小権限：閲覧専用（drive.readonly）。書き込みや削除は行いません

■ 使い方
1. Google ドライブで、対象のフォルダ（または HTML / Markdown ファイル）を開く・選択する
2. 画面の「Web プレビュー」ボタンをクリック
3. 新しいタブにページが表示されます
   （フォルダの場合は index.html を表示。無ければ先頭の html / md / txt を自動で開きます）

■ プライバシー
ファイルの内容や認証情報を外部サーバーへ送信しません。アクセスは閲覧専用で、
Google アカウントの「アプリの連携」からいつでも解除できます。
```

## 「プライバシーへの取り組み」タブ 記入内容（公開ブロッカー解消）

### 単一用途の説明（Single purpose）

```
Google ドライブ上に保存された HTML/CSS/JavaScript・画像等で構成された Web ページを、ブラウザ内でレンダリングしてプレビュー表示すること。これが本拡張機能の唯一の目的です。
```

### 権限の正当性

- **identity**
  ```
  ユーザーの Google ドライブ上のプレビュー対象ファイルにアクセスするため、chrome.identity で Google の OAuth アクセストークンを取得します。サインインとファイル取得の認可にのみ使用します。
  ```
- **storage**
  ```
  プレビューセッションの状態（対象フォルダ・エントリファイル名など）とユーザー設定をデバイス内に保存・復元するために使用します。Manifest V3 の Service Worker が再起動した後も動作を継続させるために必要です。
  ```
- **ホスト権限 `https://www.googleapis.com/*`**
  ```
  プレビュー対象ファイルを取得するため Google Drive API へ通信します。通信先は Google の API に限られ、ファイル内容を第三者サーバーへ送信することはありません。
  ```

### リモートコードの使用

- **「リモートコードを使用していません」を選択**する。
  本拡張機能はすべてのスクリプトをパッケージに同梱（esbuild でバンドル）しており、外部からのコード読み込み・実行は行わない。
  （将来 drive.file ＋ Google Picker に移行すると Picker が外部スクリプトを読むため、その時は「使用している」に変更し理由を記載する）

### データ使用に関する表明（認証チェック）

- 収集・外部送信するユーザーデータ：**なし**（ファイル内容・トークンはブラウザ内で完結し、通信先は Google のみ）
- 次の3項目すべてにチェック：
  - 承認された使用以外に第三者へ販売・譲渡しない
  - アイテムの単一用途と無関係な目的で使用・転送しない
  - 信用力の判断や融資目的でのデータ使用・転送をしない

## 設定（公開前に必須）

- パブリッシャーの**連絡先メールアドレスを入力 → 確認メールのリンクで認証**（ダッシュボードの「設定」ページ）
